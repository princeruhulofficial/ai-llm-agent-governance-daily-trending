# 🤖 AI LLM Agent Governance Daily Trending

Daily automated tracking of **trending GitHub repositories** in:

- **AI Agents** & Agentic AI systems
- **LLMs** (Large Language Models) & Foundation Models  
- **AI Governance**, Safety, Ethics, Alignment & Responsible AI

## 🌟 How it works

Every day, Grok AI automatically:

1. Searches GitHub for the hottest repositories in the above categories (recent activity + high popularity).
2. Compiles a beautiful curated report of the top trending ones.
3. Creates a new **GitHub Issue** titled with the date.
4. Sends you an email notification with the summary.

Find all daily reports in the **[Issues](https://github.com/princeruhulofficial/ai-llm-agent-governance-daily-trending/issues)** tab.
